Build

it has gulp file. You should use it. Here few steps:

  1. go to root directory
  2. type npm install
  3. bower install
  4. gulp bower to inject bower-dependencies
  5. gulp watch to watch for sass files and compile them
  6. gulp deploy to make a minified version of all css and js, it will create separate directory “dist”

Also you need node.js/ruby and sass installed globally to make it work

Dev and Dist version

Attached archive has two big folders (dev and dist):

Dist folder is a result of gulp deploy, which has all files concatinated and minified

Dev folder is a source folder

Javascript files

\app\js\main.js - pretty much everything is here
\app\js\plugins.js - plugins inits (owl-carousel, jcarousel)

Sass/CSS files

\app\sass - general styles
\app\sass\templates - modules (pages)

\app\css\bundle.min.css - result of gulp sass task (all sass files concatinated and minified)
